Array Metodlari Qo'llanmasi

JavaScript array metodlarini o'rganing va misollarni ko'ring.

map()

map() metodi arrayning har bir elementiga funksiya qo'llaydi va natijada yangi array qaytaradi.

Misol:


const sonlar = [1, 2, 3, 4];
const kvadrat = sonlar.map(son => son * son);
console.log(kvadrat); // [1, 4, 9, 16]

filter()

filter() metodi arraydan testni muvaffaqiyatli o'tgan elementlarni tanlab oladi va yangi array yaratadi.

Misol:


const sonlar = [1, 2, 3, 4];
const juft = sonlar.filter(son => son % 2 === 0);
console.log(juft); // [2, 4]

reduce()

reduce() metodi array elementlariga funksiya qo'llaydi va bitta yakuniy qiymat hosil qiladi.

Misol:


const sonlar = [1, 2, 3, 4];
const summa = sonlar.reduce((jam, son) => jam + son, 0);
console.log(summa); // 10
